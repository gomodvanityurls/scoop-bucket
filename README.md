# scoop-bucket

Scoop bucket for [gvu](https://github.com/gomodvanityurls/gvu) — the CLI client for Go Module Vanity URL service.

## Usage

```powershell
scoop bucket add gomodvanityurls https://github.com/gomodvanityurls/scoop-bucket
scoop install gvu
```

## Update

```powershell
scoop update gvu
```
